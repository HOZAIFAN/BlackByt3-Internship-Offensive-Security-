Scanvora User Guide

Your complete, plain-English guide to running Scanvora — no tech skills needed.

v5.1 — Production Ready

Before You Begin

Requirements
  • Python installed on your computer
  • An internet connection
  • A domain you own or have written permission to scan

That's it. No other experience needed.

1
Install Scanvora

Open your terminal (or Command Prompt on Windows) and run:

pip install aiohttp aiodns
Note
You only need to do this once. After the first install, skip straight to Step 2.
2
Run the Wizard

Navigate to the folder where scanvora_wizard.py is saved, then run:

python scanvora_wizard.py

You will see a welcome screen. From here, Scanvora guides you through the rest.

3
Answer the Questions

Scanvora asks you a series of simple questions. Here is what each one means and what to type.

Question 1 — Target Domain
Enter the domain to scan (e.g., example.com):

What to type: The website you want to scan.

Example: google.com or mybusiness.org

  • Do not include https:// or www.
  • Correct format: example.com
  • Wrong format: https://www.example.com
Question 2 — Scan Mode
Select scan mode:
  1. Full Recon
  2. Fast Scan
  3. Brute Force Only
  4. Passive OSINT

What to type: A number from 1 to 4.

If you want...ChooseTime
The most complete scan110–30 min
Quick results21–5 min
To use your own wordlist35–20 min
To stay completely undetected4Under 2 min
Not sure?
Type 1 for Full Recon — it is the most thorough option.
Question 3 — Wordlist (Brute Force Mode Only)
Enter path to wordlist file (or press Enter to skip):

This question only appears if you chose Brute Force Only mode.

What is a wordlist? A plain text file with common subdomain names, one per line:

admin
mail
api
dev
staging
  • If you have a wordlist: type the full file path, e.g. /home/user/wordlist.txt
  • If you do not have one: press Enter to skip
Question 4 — Number of Workers
Number of concurrent workers [default: 25]:

What does this mean? How many scans run at the same time. More workers means faster results but uses more of your computer's resources.

  • Press Enter to use the default — recommended for most people
  • Type 10 for a slower, safer scan
  • Type 50 for a faster scan on a powerful machine
Tip
If the scan keeps timing out, lower this number to 10 and try again.
Question 5 — Rate Limit
Requests per minute [default: 100]:

What does this mean? How many requests Scanvora sends each minute.

  • Press Enter for the default — works in most situations
  • Type 50 to go slower and avoid triggering the target's defenses
Question 6 — API Keys (Optional)
Enter Shodan API key (or press Enter to skip):
Enter SecurityTrails API key (or press Enter to skip):
Enter VirusTotal API key (or press Enter to skip):

What are these? Premium services that give Scanvora access to extra data about the target.

  • If you have a key: paste it in and press Enter
  • If you do not: press Enter to skip — the scan still works

Before the scan starts, you will see a summary screen:

Start scan? (yes/no):
  • Type yes to begin the scan
  • Type no to go back and change a setting
4
Wait for the Scan

Once confirmed, the scan begins and progress updates appear on screen.

Important

5
View Your Results

When finished, two report files appear in the same folder as the script:

scanvora_example.com_2024-01-15_14-30.json
scanvora_example.com_2024-01-15_14-30.txt

Open the .txt file first — it is the easiest to read:

SCANVORA REPORT
Target: example.com
Date: 2024-01-15

DISCOVERED SUBDOMAINS (12 found)
----------------------------------
mail.example.com       -> 192.168.1.1   [PRIORITY: HIGH]
api.example.com        -> 192.168.1.5   [PRIORITY: MEDIUM]
staging.example.com    -> 192.168.1.8   [PRIORITY: LOW]

Open the .json file if you want to import the data into another security tool.

Common Questions

Can I stop the scan early?
Yes. Press Ctrl + C on your keyboard. The scan will stop safely.
The scan seems frozen. Is it broken?
Not necessarily. Some scans take a long time. Wait a few minutes. If nothing happens after 5 minutes, press Ctrl + C, then run again with fewer workers — try 10.
I got a "permission denied" error.
Make sure you are allowed to scan that domain. If you used a wordlist, double-check the file path is correct.
Where are my report files?
In the same folder where scanvora_wizard.py is saved.
Can I scan multiple domains?
Yes. After one scan finishes, run python scanvora_wizard.py again and enter a different domain.
Do I need API keys?
No. API keys are optional. Scanvora works fine without them.
What if I typed the wrong domain?
At the confirmation screen, type no to go back and correct it.

Tips for Best Results

  • Start with Fast Scan if you just want a quick overview
  • Use Full Recon when you need complete coverage
  • Lower your worker count if you are on a slow or unreliable network
  • Add API keys if you want richer data in your reports
  • Read the .txt report first — it is the most human-readable format
